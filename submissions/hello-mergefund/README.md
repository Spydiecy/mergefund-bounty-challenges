## 🧠 How It Works

### Blockchain Implementation

The application implements a simplified blockchain with:

- **Blocks**: Each containing transactions, timestamps, and cryptographic hashes
- **Mining**: Proof-of-work algorithm that finds valid hashes with leading zeros
- **Transactions**: Special transaction containing the "Hello MergeFund" message
- **Validation**: Hash verification to ensure blockchain integrity

### Mining Process

1. Creates a new block with a "Hello MergeFund" transaction
2. Attempts to find a valid hash by incrementing the nonce
3. Updates UI with real-time mining statistics
4. Reveals the message when a valid block is found
5. Adds the block to the blockchain history
